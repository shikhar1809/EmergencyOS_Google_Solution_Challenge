# Pilot study — design template

## Objectives

- Measure **time-to-SOS-completion**, **task success**, and **perceived usability** during a controlled pilot (e.g. Lucknow-focused demo).  
- Compare **untrained civilians**, **first-aid trained users**, and **clinicians / responders** (if available).

## Participants

- Target **15–20** participants per wave (adjust for ethics / IRB constraints).  
- Record demographics only as allowed by your consent form.

## Tasks

1. *“Your friend collapses with chest pain—use the app to get help.”*  
2. *“You witness a road accident with bleeding—complete SOS and open guidance.”*  
3. *“You are a volunteer—accept an incident and interpret the map.”*  

## Metrics

| Metric | Operational definition | Target |
|--------|-------------------------|--------|
| SOS completion time | Start → incident created | < 30 s (median) |
| Time to first guidance | SOS → AI / Lifeline visible | < 45 s |
| SUS | System Usability Scale questionnaire | ≥ 70 (exploratory) |
| Errors | Mis-taps, abandon rate | Track qualitatively |

## Qualitative

- Semi-structured interview: trust, clarity, language, offline anxiety.  
- Log **quotes** (anonymized) for the Solution Challenge story.
